Python2.7安装MySQL-Python

直接执行pip命令

报错_mysql.c(42) : fatal error C1083: Cannot open include file: ‘config-win.h’: No such file or directory

安装mysql-connector-c

url: https://dev.mysql.com/downloads/connector/c/6.0.html

下载相应的版本，比如64位的windows则下载：

mysql-connector-c-6.0.2-winx64.msi

默认安装即可
